==== Russian ====
ei_maper - редактор уровней для игры Проклятые Земли.
В настоящий момент представлен в виде Демо-версии, умеет:
- открывать\закрывать *mpr, *mob файлы.
- передвигаться по карте
- выделять и удалять объекты.

Для корректной работы необходимо указать в опциях путь Figure Path(1) до figures.res (отсюда подтягиваются модели), а также Texture Path(1) до textures.res (отсюда подтягиваются текстуры).

Внимание! чтобы управление работало, нужно переключиться на английский язык.
Управление (сделано по аналогии со старыми версиями Blender):
передвижение - w,a,s,d (при зажатом shift ускоряет передвижение в 2 раза)
вертикальный подьъём - e
вертикальный спуск - q
вращение камеры - СКМ
выделение объектов - ПКМ (через shift - множественное выделение)
удаление объектов - Del
выгрузка(удаление) всех объектов - u
выгрузка(удаление) ландшафта - m

что пока не умеет делать редактор (планируется в будущем):
- отображать/редактировать параметры объектов
- редактировать скрипты
- создавать объекты
- добавлять/редактировать ландшафт(через сторонние приложения)
- сохранять *mob/*mpr файлы
список будет дополняться по мере поступления идей

====English====
ei_maper - it's a level editor for Evil Islands game.
In the demo version, it can:
- open \ close * mpr, * mob files.
- move around the map
- select and delete objects.

For correct operation, you must specify in the options the path "Figure Path (1)" to figure.res, as well as "Texture Path (1)" to textures.res.

Attention! for the key bindings to work, you need to switch keyboard layout to English.
Key bindings (made by analogy with older versions of Blender):
movement - w, a, s, d (with a clamped shift, accelerates movement by 2 times)
move up - e
move down - q
camera rotation - MMB
selection of objects - RMB (via shift - multiple selection)
deleting objects - Del
unload mob - u
unload landscape - m
